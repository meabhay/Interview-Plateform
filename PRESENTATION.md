# Slide 7: System Architecture

A high-level overview of our modern tech stack.

1.  **Frontend (Client)**: A user interacts with the web app built on **Next.js**, **React**, and **Tailwind CSS**.
2.  **Voice Interaction**: The **Vapi.ai** service manages the real-time, streaming voice conversation.
3.  **Backend Logic**: **Next.js Server Actions** receive requests, process data, and interact with the database via the **Prisma ORM**.
4.  **AI Core**: After the interview, the transcript is sent to **Google Gemini** for analysis, scoring, and feedback generation.
5.  **Database**: All data, including user profiles, interview history, and feedback, is stored in a **PostgreSQL** database hosted on **Supabase**.
6.  **Authentication**: User sign-in is securely handled by **NextAuth.js** with Google OAuth.
